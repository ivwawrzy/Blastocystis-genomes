# Blastocystis- subtypes 1 and 8 genomics project

In this project we sequenced and annotate a Blastocystis ST1 and 8.
You will find in the repository genbank, the gbf files, and in the repository transposable_element the result of the analysis with the script One code to find them all (https://doua.prabi.fr/software/one-code-to-find-them-all) and the set of transposable element use for identify the TE.
